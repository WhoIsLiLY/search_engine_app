Install library dan modules yang dibutuhkan oleh proyek
1. npm install
2. pip install -r requirements.txt

Setelah instalasi selesai, jalankan perintah berikut di instance terminal yang berbeda
1. php -S localhost:3000
2. npm run build

Link ke GITHUB untuk repo yang terupdate: https://github.com/WhoIsLiLY/search_engine_app.git

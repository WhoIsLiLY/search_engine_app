1. Untuk mengunduh semua library Python yang dibutuhkan, buka terminal dan jalankan perintah: pip install -r requirements.txt
2. .gitignore digunakan untuk mencegah file atau directory yang TIDAK BOLEH dipush ke dalam repository github
3. .env adalah variable lingkungan yang digunakan untuk menyimpan berbagai variable yang dibutuhkan oleh proyek. 
Beberapa isi variable TIDAK DIPERBOLEHKAN untuk dipush ke dalam github, itulah mengapa .env masuk ke dalam gitignore.
